# **モンスターボウル**/MonsterBowl
持ちながら動物に向かって右クリックすると捕獲することのできる代物
<br>名札や交易内容とかも保存されるから意外と便利
<br>※モンスタ〇ーボールとは一切関係ありません

### 作り方
普通の[木のボウル](http://www.minecraftwiki.net/wiki/Bowl)を作って`モンスターボール`と金床等で名前を付けます
